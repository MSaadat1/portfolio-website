The Space Tourism Website

Traveling to Space

This is a Space Tourism Website that I build it from Scratch. In this website I used most important elements of HTML5, deployed grid and flex box in CSS. To use it ,I also add Java Script. I used custom CSS so it would be easy if I like to change any color, text or formate of it in the future.

This website consist of four wibsite pages. They are the 00 Home, 01 Destination, 02 Crew, 03 Technology. All sites have navbar with the link that can be connect to any page you like. The Destination page has four buttons and in each button an image and datails can be shown once click. In the crew page also buttons are used but with dot style that have images and information about each crew memebers and finally the Technology site. This site has number button with hover effect.

User can view the optimal layout for each of the website's pages depending on their devices' screen size. Also, User see hover states for all interactive elements on the pages. Moreover each page viewed and be able to toggle between the tabs to see new information.
